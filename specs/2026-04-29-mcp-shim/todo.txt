  A. Fix the layering smell — the move

  1. Move the MCP runtime into the server package. Relocate cli/src/mcp/{agent-identity,logger,tool-logging,tools/*} → server/src/mcp/*, and pull along bash/*, content/enrichment.ts, content/shadow-log.ts, plus the Config schema + resolveContentDir/resolveLockDir helpers +
