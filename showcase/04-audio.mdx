# Audio

Pure HTML5 `<audio>` wrapper. Self-closing leaf. `controls` is now an explicit prop (default true) — set `controls={false}` for chrome-less playback (rare, mostly for hero loops).

## Autoplay (muted)

Autoplay with audio is blocked by browsers. Pair with `muted` for reliable autoplay.
