# Accordion

Standalone HTML5 `<details>` substrate. Either form parses to the same descriptor: native `<details>` is the markdown form, `<Accordion>` is the MDX JSX form. Cross-browser exclusive grouping via HTML5 `<details name="…">` (Chrome 120+, Safari 17.2+, Firefox 130+).

## HTML5 `<details>` form (markdown form)

<details open>
<summary>HTML5 native — defaultOpen</summary>

Body content can be any block-level markdown:

- Lists
- **Bold**, _italic_, `inline code`
- Links: [CommonMark §5.5.3](https://spec.commonmark.org/0.31.2/#raw-html)
  </details>

<details>
<summary>HTML5 native — closed</summary>

Click the summary to expand.
</details>

## MDX JSX form

<Accordion title="MDX JSX Accordion">
  Same descriptor as HTML5 `<details>`; `title` maps to `<summary>`.
</Accordion>

<Accordion title="defaultOpen" defaultOpen>
  Starts expanded on first render.
</Accordion>

## With description + icon

<Accordion
  title="Full superset"
  description="A longer multi-line description surfaces below the title"
  icon="lucide:settings"
  defaultOpen
>
  The full 6-prop superset: `title` (required), `defaultOpen`, `icon`, `description`, `id`, `name`.
</Accordion>

## Exclusive grouping via `name`

Multiple accordions sharing a `name` act as a radio group — opening one closes the others (HTML5 `<details name>` behavior).

<Accordion title="Group A — Item 1" name="faq-group-a">
  Part of group `faq-group-a`. Opening this closes the other group-a items.
</Accordion>

<Accordion title="Group A — Item 2" name="faq-group-a">
  Part of group `faq-group-a`. Browser-enforced exclusivity.
</Accordion>

<Accordion title="Group A — Item 3" name="faq-group-a">
  Part of group `faq-group-a`. Click to open — closes the others above.
</Accordion>

## Anchor links via `id`

An `id` on the Accordion lets you deep-link directly to it (`#my-accordion` in the URL).

<Accordion id="getting-started" title="Getting started section">
  With `id="getting-started"`, this accordion can be linked to directly — click to hear the hash change in the URL bar.
</Accordion>

## Nested content

Accordions have `content: 'block*'` — they can hold other descriptors.
