# D5d — TikTok / Reels / Shorts (Silent Autoplay, Mobile-First)

> **Lens.** Vertical 9:16. 15-60 sec hard ceiling, 7-15 sec sweet spot. SILENT autoplay default — captions do the talking. The first 1 second decides whether the thumb stops or swipes. The audience is Gen Z curious-about-AI, bootcamp students, self-taught devs, dev-content creators (Fireship, Theo, Primeagen-adjacent), and "look at this cool thing" doomscrollers. Different physics than X/HN: no headline craft, no comment-thread credibility — purely visual hook + caption + remixability. Designed to be **stitch-bait** so creators amplify for free.
>
> **Cold-open discipline.** Frame 1 is everything. A blank editor is death. Frame 1 must contain motion, a number, a familiar logo being subverted, a face, or a typed sentence already half-finished.

## The 18 ideas

### Tier 1 — Pure Bolt-archetype (visible AI doing visible work)

**1. "I typed nothing."**
*Cold open (frame 1):* a blinking cursor in an empty wiki page, with a caption already on screen: **"i typed nothing."**
*0:01-0:12:* Text starts streaming in by itself — full markdown article on, say, "How OAuth works" — headings, code fences, a wiki-link `[[JWT]]`, a Mermaid diagram rendering live. Bottom caption sticky: **"my AI is writing my wiki rn."**
*0:13:* Cut to the file tree — three new files appeared (`oauth.md`, `jwt.md`, `pkce.md`). Caption: **"and it linked them."**
*0:14:* End frame: just the OK logo + `bunx open-knowledge`. No CTA voiceover. Let the comment section ask "wait what is this."

**2. "Cmd+Z the AI"**
*Cold open:* split-screen editor showing one paragraph that's clearly human ("my notes from today's meeting…") and one paragraph that's been color-shaded as agent-written.
*0:01-0:08:* hand presses Cmd+Z four times. Each press: only the agent paragraph rolls back, line by line. The human paragraph never flinches.
*0:09:* caption explodes onscreen: **"Cmd+Z deletes the AI. not your work."**
*0:10:* fade. No further explanation. Pure mechanical magic trick — comprehensible without sound or context.

**3. "watch me race the AI"**
*Cold open:* timer in top-right counts up from 0:00. Split-screen editor — left is "me typing", right is "Claude typing".
*0:01-0:30:* both write the same article ("Postgres indexing tips"). The agent finishes first, complete with code blocks and a `[[B-tree]]` wiki-link. Human side has 2 typos and the middle of a sentence.
*0:31:* caption: **"the wiki has both versions saved. i can pick lines from each."** Cut to the diff/version timeline showing both authors' commits color-coded.
Format primed to be **reaction-stitched** — every dev creator's instinct is to argue with it.

**4. "POV: you went to bed and Claude maintained your wiki"**
*Cold open:* phone-clock graphic showing 2:14am. Editor open, dark mode.
*0:01-0:25:* timelapse of the version-history sidebar — commits stack up, color-coded human/agent (mostly agent overnight). File tree expands. Backlinks count ticks. Graph view spins to add new nodes.
*0:26:* phone-clock 7:31am. Caption: **"woke up to 47 new pages and a refactored index."**
*0:27:* the agent has left a `[[Today's summary]]` redlink — click it, page exists, has bullet list of what changed. Cut. **GenZ "while you were sleeping" framing.**

### Tier 2 — Toggle / view-flip magic tricks (10-15 sec, pure visual)

**5. "the WYSIWYG flip"**
*Cold open:* a beautiful rendered article — heading, callout box, code with syntax highlight, an embedded diagram.
*0:01:* one keypress (visualized as a key cap pulsing). The whole view dissolves into the raw markdown source — `## H1`, ```` ```ts ````, the same content, identical Y position. A second key press, back to rendered. Three flips total in 8 seconds, perfectly synced to a trending audio's beat (caption says **🔇** so people unmute for the sync).
*Caption sticky:* **"both views. same file. same cursor."**
*The hook:* this is what Notion users have been begging for since 2018.

**6. "the graph view spin"**
*Cold open:* a single page floating in dark space.
*0:01-0:08:* every wiki-link the user types `[[ ]]` adds a new node and an edge that physics-snaps into place. The graph grows to ~40 nodes and starts gently rotating like a galaxy.
*0:09:* caption: **"every `[[link]]` you type is a node. forever."**
*Pure aesthetic.* No CTA. Designed to be saved to "wallpaper" boards. The graph IS the content.

**7. "diff vs entire-file rewrites" (Notion drag)**
*Cold open:* split-screen. Left labeled "Notion AI". Right labeled "Open Knowledge".
*0:01-0:08:* user prompts "make this clearer." Notion side: entire page disappears, then re-types from scratch — old version gone. OK side: only the two muddled sentences highlight, get replaced inline. Five seconds. Caption: **"only changes what it should."**
*0:09:* freeze frame. End. Comparison post baked into frame 1.

### Tier 3 — Avatar / personality / collection-loop content

**8. "show me your KB animal"**
*Cold open:* a row of cute animal avatars (Bird, Cat, Dog, Fish, Rabbit) wiggling — that's our existing presence-avatar set.
*0:01:* caption: **"your name → your animal. forever."**
*0:02-0:10:* type-test screen — "Sarah" → Cat. "Miles" → Bird. "Andrej" → Fish. "type your name in the comments and i'll reply with your animal."
*0:11:* end card with the 5-animal grid + name spelling.
**Comment farming machine.** Every commenter gets a deterministic emoji reply, generated by a comment-bot. This is the cheapest, highest-ROI clip type that works specifically because TikTok rewards comment volume.

**9. "the agent has an animal too"**
*Cold open:* the presence bar — Cat avatar (human) sits next to a Claude-orange circle (agent).
*0:01:* caption: **"my collaborators."**
*0:02-0:08:* type a sentence; Cat's edits flash green. Wait two seconds; Claude-circle's edits flash orange in the same paragraph. Both cursors visible simultaneously. Almost no other movement — the *quietness* of the duet is the hook.
*0:09:* caption flips: **"this is local. no cloud. it's just on my mac."**
**Distinctive because every other AI tool clip is loud and chaotic; this one is intimate.**

**10. "if your KB had a face"**
*If* we ship a mascot/Tamagotchi (per the `/buddy` precedent in the worldmodel) — pure aesthetic shot of the mascot reacting to events. Mascot blinks when agent writes. Mascot sleeps at midnight. Mascot does the "👀" emote when you and Claude edit the same paragraph at once.
*Caption:* **"my wiki has a pet."** Pair with the nostalgic-Tamagotchi audio trend already on TikTok.
**Wildly stitch-able.** Every "show me your buddy" reply is free reach.

### Tier 4 — Speed / install / "look how easy" (the bolt.new template)

**11. "12 seconds, install to editor"**
*Cold open:* a stopwatch on a black screen reading 0:00 — the *only* thing on frame 1. Familiar visual language (Theo / Fireship style).
*0:01:* terminal appears, types `bunx @inkeep/open-knowledge` itself. Stopwatch ticks.
*0:02-0:11:* boxed banner appears (~5s in), browser opens (~9s), editor loads (~11s).
*0:12:* stopwatch freezes at 12s. Caption: **"local AI wiki. no signup. no cloud."**
*Numbers-in-thumbnail* is a proven TikTok format. The clip lives or dies on whether 12 seconds is real on a fresh machine — must be tested.

**12. "from `bunx` to first agent edit in under a minute"**
*Cold open:* split timer + terminal. 0:00.
*0:01-0:55:* terminal install → editor open → user types "create a project doc for my todo app" in Claude Code (other window) → tab back to editor → page is being written live.
*0:56:* caption: **"local-first. fully open source. zero tokens billed."**
*0:57:* end. **Demonstrates the actual moat (agent-native + local) in <60s with no narration.**

### Tier 5 — Cultural / scenario clips that sell the lifestyle

**13. "POV: every meeting note auto-becomes a wiki page"**
*Cold open:* iPhone voice-memo screen, recording.
*0:01-0:04:* zoom into the laptop next to it — Claude Code window says "transcribed; staging draft." Editor pops open. New page `2026-04-14-meeting.md` appears in sidebar with frontmatter, action items, and `[[Sarah]]` `[[Q3-roadmap]]` wiki-links auto-extracted.
*0:05:* user clicks on `[[Q3-roadmap]]` — page already exists with a backlink reading "mentioned in 2026-04-14-meeting".
*0:06-0:08:* caption: **"the agent does the bookkeeping. i just talk."**
**Reframes the product from "wiki tool" to "agent-native life infrastructure."** GenZ "what i wish i had in college" framing.

**14. "i made a wiki for my bootcamp notes"**
*Voiceover-optional* (works silent with captions). First-person.
*Cold open:* messy desktop — 47 untitled .md files in a folder.
*0:01-0:25:* user runs `open-knowledge`, points it at the folder. Sidebar populates. Asks Claude (in Claude Code panel) to "tag and link these by topic." Watch agent edit each file in real-time, presence cursor visible. Frontmatter gets tags. Wiki-links appear. Graph view fills out.
*0:26:* end frame: graph view, ~30 nodes, beautifully clustered.
*Caption:* **"my notes from 12 weeks of bootcamp. now they link to each other."**
**Solo-author / learning-in-public archetype** — the highest-engagement format on dev TikTok.

**15. "the side-by-side"**
*Cold open:* split-screen. Left: Obsidian, single cursor. Right: Open Knowledge, two cursors (human + Claude). Same vault, same file open.
*0:01-0:06:* both editors get typed into. Obsidian only shows one author's keystrokes. OK shows both authors' keystrokes interleaving in the same paragraph in real-time.
*0:07:* caption: **"spot the difference."**
*0:08-0:10:* arrow appears pointing at the second cursor. Caption: **"that's the AI. typing live. in your wiki."**
**Doesn't dunk on Obsidian** — frames us as "Obsidian + this one extra thing." Obsidian fans are the primary share-vector.

### Tier 6 — Stitch-bait / creator-hook (for influencer remix amplification)

**16. "name a feature i should add"**
*Cold open:* user's face on camera (not required if we don't have a face; can be Claude-circle avatar). Caption: **"name a feature, i'll add it to my wiki tool by tonight."**
*0:01-0:15:* live build moment — agent writes the feature in front of you. Pure stitch-bait. Designed to seed a recurring series. Each top comment becomes a follow-up clip → algorithmic snowball.
**This isn't a one-shot clip; it's a content engine.**

**17. "Open Knowledge in 100 seconds" (Fireship template)**
Drop the **raw demo asset** as a downloadable b-roll pack on the launch site. Includes: install timelapse, two-cursor edit, WYSIWYG flip, graph spin, agent overnight timelapse, version-history scrub. **Email Fireship a pre-edited rough cut** the day before launch — his "X in 100 seconds" slot is the single most efficient TikTok-pipe in dev media. Cost: zero. Even if he doesn't take it, the b-roll pack lets micro-creators (1K-50K follower devs) each produce their own clip without filming. Distribute via the launch repo's `media/` folder.
*Cold open of the official version (if Fireship doesn't bite, we publish ourselves):* the OK logo morphs into the Obsidian logo, then back, then morphs to a Notion logo, then back, then to a folder of `.md` files, then back. Caption: **"all of this. in 100 seconds."** Then full Fireship-rate narration with captions burned in.

### Tier 7 — Emotional / aesthetic (saves not likes)

**18. "the dark-mode wiki page that's just pretty"**
*Cold open:* a beautifully typeset wiki page in dark mode — heading hierarchy, a Mermaid diagram, a callout, a code fence with syntax highlight, a pull quote, a `[[link]]` underlined. Soft focus glow. A subtle ambient gradient.
*0:01-0:10:* slow camera pan (CSS scroll). No edits. No agent. Just the page existing. Lo-fi audio (not voice).
*Caption (one-liner, late):* **"text can be beautiful again."**
*Saves > Likes.* This is a "design TikTok" play — it gets shared to interior-design / studyblr / coding-aesthetic boards, which loop back into dev creators.
**The aesthetic wing is a parallel funnel** to the demo wing.

## WILD CARDS

### WC1 — "Run my Wiki" (the ambient-stream Twitch/TikTok-LIVE hybrid)

**Concept.** A 24/7 always-on TikTok LIVE / YouTube LIVE / Twitch stream of an Open Knowledge editor sitting open. A community-controlled Claude Code session, fed by viewer comments, continuously edits, refactors, links, and tends a fully public wiki on a real subject (e.g. "wiki of weird Wikipedia articles," "wiki of indie games," "wiki of every Pokémon card"). Viewers watch the agent typing in the editor. Presence avatars show **viewer count** as anonymous animal icons clustered in the sidebar.

**Why TikTok specifically:** TikTok LIVE is biased toward "ambient watch" content — slime videos, bonsai cams, terrarium streams. A wiki being maintained by a benevolent AI overnight, with viewers chiming in via chat (each chat message → an MCP tool call to "add this fact" → staged as a draft → the maintainer-character merges or rejects), produces hours of low-key content. The clipped highlights are infinite raw material for shorter clips.

**Mechanic.** Subscribers/donations gate write-permission tiers — `proposer` (everyone), `editor` (subscribers), `maintainer` (mods). Maps to the actual S4 permission model. The product **demonstrates itself** as the native UI of the stream.

**Risk.** Moderation. Agent abuse. Cost of one continuous Claude session. But the *concept* — a wiki you can watch grow — is genuinely novel and screenshots from it become organic clips on their own.

### WC2 — "the AI made me a TikTok of my own wiki"

**Concept.** Ship `open-knowledge clip <page>` — a CLI command that takes one wiki page and renders a 15-second 9:16 vertical video of it: dark mode, the page typing itself out at viewer-comprehensible speed, the wiki-links lighting up as little popovers, the graph view spinning at the end with that page highlighted, the OK logo last.

Output: an `mp4` ready to upload to TikTok. Watermarked with `open-knowledge.dev`. Optional auto-caption from the page's title/frontmatter.

**Why this is the play.** Every user of Open Knowledge becomes a content creator for Open Knowledge — automatically — and the clip is *about their content*, which they actually want to share. We bypass the "make a demo video for our tool" creator block entirely. Each user's page-of-the-week becomes a TikTok with our watermark.

Pair with a **`#myKB` hashtag** at launch and a leaderboard of the prettiest auto-generated clips. The viral asset is the *user's content rendered through us* — same primitive Bolt used (every generation is a tweet-able URL); we extend it to short-form vertical video.

**Cold open template baked in:** every clip starts with the title in huge type appearing one character at a time on a black screen — proven 1-second-hook pattern.

### WC3 — "the silent ASMR debugger"

**Concept.** Lean fully into the GenZ ASMR/lo-fi-aesthetic axis. Long-form (3-10 min) silent video: a single Open Knowledge editor, dark-mode, soft ambient lighting visible in the screen recording's reflection (i.e. shot on a real laptop, not screen-capture). The agent slowly, carefully, refactors a messy folder of notes into a beautiful wiki — taking 10 minutes to do it at "human watching speed". Background: the *only sound* is the soft typing of the keys (recorded foley) and the chime when a backlink resolves. No music. No voiceover.

**Why this works:** taps the existing **"productivity ASMR"** subgenre (notion-girls, bullet-journal lapses, code-typing lo-fi). The audience is huge, mostly female, mostly study-aesthetic, currently *not* served by dev-tool marketing. Open Knowledge — quiet, two-cursor, gentle agent, wiki-graph aesthetic — is the only AI dev tool that *fits* the visual grammar. Most AI tools are too loud (terminals streaming! progress bars! GPU fans!). Ours can be still.

**Distribution:** drop on TikTok, YouTube, AND repurpose as a 60-second loop on the docs-site hero. A muted 10-minute lo-fi clip of *our product working* is a marketing asset that no competitor can authentically produce.

## Cold-open audit (the single most important variable)

Best frame-1 hooks ranked:

1. **A blinking cursor with a half-typed caption** (#1, #2). Universally legible. Zero context required.
2. **A timer counting up from 0:00** (#11, #12). Numbers stop scroll.
3. **A face or animal avatar making eye contact** (#8, #9, #10). Avatars do "face" work for us without a real human.
4. **Familiar logo morph** (#17 Fireship-cover open). Pattern-interrupt.
5. **Split-screen with a familiar vs ours** (#7, #15). Comparison primes argument.
6. **Beautiful aesthetic shot** (#18, WC3). Saves over likes; algorithmic compounding.

Worst frame-1: a logo, a CLI banner alone, an empty editor. **Never open with our brand.**

## Caption style guide (for silent autoplay)

- Lowercase. **GenZ does not capitalize.**
- Max 6 words on screen at once
- Sticky bottom-third caption persists; floating top caption changes
- One-emoji-max per caption (🔇 for "unmute me" cue is allowed twice)
- End every clip with a question or a half-statement so the comment section finishes it ("…and yes it's open source" implied)
- **Never put a URL in the caption.** TikTok suppresses external links. Bio link only.

## Track-of-content roll-out plan (3 weeks, cheap)

**Week -1 (pre-launch):** Drop 3 ambient/aesthetic clips (#6, #18, WC3) with no product reveal. Build account follower base + algorithm signal. Captions tease "something coming."

**Week 0 (launch day):** Drop the headline clip (#1 "I typed nothing") + a comparison clip (#7 or #15) + the install-speed clip (#11). Three at 9am/1pm/7pm ET. Pin #1 to top of profile.

**Week +1:** Stitch-bait (#16) + side-by-side (#15) + meeting-notes scenario (#13). React to top comments with reply-clips of the named-in-comments features.

**Week +2:** Distribute b-roll pack to 20 micro-creators (1K-50K) with a personal note. Ship `open-knowledge clip` (WC2). Hashtag launches `#myKB`. Begin the 24/7 stream (WC1).

Total estimated production cost: ~30 hours of edit work (one editor, one week). Most clips are screen-recordings + caption overlays. The only expensive pieces are WC1 (operations) and WC3 (foley + lighting).

## What this lens uniquely contributes

X/HN ideation focuses on **headlines**. TikTok ideation forces three things the other channels miss:

1. **Frame-1 discipline** — every other channel hides behind a title; here you have one frame to earn the watch.
2. **Comprehension without sound or context** — forces clarity in product surfaces themselves (if the WYSIWYG flip isn't visually obvious in 8 seconds, *the product UI* needs work, not the marketing).
3. **The user-as-content-creator loop** — WC2 (`open-knowledge clip`) is a feature surfaced by this lens that wouldn't exist if we only thought about X/HN.

The platform-fit is real: Bolt's TikTok ("I built Airbnb in 30 seconds") proved AI-builds-a-thing clips dominate. Open Knowledge has an even tighter version of that demo — **two cursors editing the same paragraph** — which is more visually distinctive (and harder to fake) than any text-to-app demo. The asset is sitting on the disk waiting to be filmed.
