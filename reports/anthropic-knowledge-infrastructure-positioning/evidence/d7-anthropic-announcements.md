## Findings

### Finding: Anthropic has published three canonical pieces relevant to agent knowledge management
**Confidence:** CONFIRMED
**Evidence:** Direct primary sources

1. **"Effective Context Engineering for AI Agents"** (anthropic.com/engineering)
   - Positions context engineering as the successor to prompt engineering
   - Advocates hybrid model: CLAUDE.md (orientation) + skills (procedures) + MCP (connectivity)
   - Key quote: "treating context as a precious, finite resource"
   - Introduces memory tool for building knowledge bases outside context window

2. **"Equipping Agents for the Real World with Agent Skills"** (claude.com/blog)
   - Frames skills as "modular expertise packages" = procedural knowledge
   - Positions skills as "decentralized, composable knowledge management"
   - Emphasizes open standard and cross-platform portability
   - Progressive disclosure as the knowledge loading architecture

3. **"Cowork and Plugins Across the Enterprise"** (claude.com/blog)
   - Plugins = skills + connectors + agents bundled
   - 11 knowledge work categories
   - Private marketplace model for enterprise knowledge distribution
   - PwC partnership for regulated industries

### Finding: Anthropic has NOT published a standalone "knowledge management for agents" manifesto
**Confidence:** CONFIRMED (negative search)
**Evidence:** Searched anthropic.com/blog, anthropic.com/engineering, anthropic.com/research

No blog post, talk, or announcement uses the phrase "knowledge management" as its primary topic. The knowledge framing is distributed across:
- Context engineering content (how agents access knowledge)
- Skills content (how to package procedural knowledge)
- Cowork content (how to distribute knowledge in enterprises)

**Implications:** Anthropic is building knowledge infrastructure through multiple product surfaces (skills, MCP, Cowork, memory tools) without explicitly branding it as a "knowledge management" play. The strategy is emergent from product decisions, not announced as a unified vision.

### Finding: "The Complete Guide to Building Skills for Claude" — enterprise knowledge packaging
**Confidence:** CONFIRMED
**Evidence:** resources.anthropic.com (PDF guide)

Anthropic published an enterprise-oriented guide framing skills as the way to package organizational knowledge for Claude. This is the most explicit "knowledge management" framing — positioning skills as how enterprises encode and distribute institutional knowledge.
