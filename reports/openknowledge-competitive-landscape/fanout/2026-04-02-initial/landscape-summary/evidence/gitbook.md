## Key Facts

- MCP server auto-generated for every published docs site (append /~gitbook/mcp)
- llms.txt and llms-full.txt auto-generated for every docs site
- GitBook Agent (beta): proactively identifies doc gaps from support conversations, Slack, GitHub issues
- AI Assistant: reimagined AI search with conversational knowledge discovery
- Adaptive content: tailored docs experience per individual user
- Closed source, commercial SaaS
- Primarily serves developer documentation and API reference use cases
- 12 MCP tools for content operations, 6 AI-powered prompts
