## Key Facts

- License: AGPL-3.0 core, enterprise features (SSO, Audit Logs, SCIM, AI) under proprietary /ee folder
- AI providers supported: OpenAI, Google Gemini, Ollama (local LLMs)
- AI features: Ask AI (content generation in editor), AI Answers (semantic search with vector embeddings)
- MCP server available for AI assistant integration
- Real-time collaborative editing using CRDT-based synchronization
- Rich content types: diagrams (Draw.io, Excalidraw, Mermaid), embedded media, file attachments, math equations
- Built as self-hosted alternative to Confluence and Notion
