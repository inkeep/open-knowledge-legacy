## Key Facts

- Open source framework for agent memory
- Scalable memory architecture that dynamically extracts, consolidates, retrieves information
- Graph-based memory representations for complex relational structures
- Performance: 26% higher accuracy than OpenAI Memory (LOCOMO benchmark), 91% faster, 90% fewer tokens
- Apache Cassandra support (v1.0.1, Nov 2025)
- Valkey support (v0.1.118, Sep 2025)
- FastEmbed integration for local embeddings (no API call needed)
- Graph memory in production by early 2026 (was experimental in 2024)
- Described as "most mature long-term memory solution in 2026" among frameworks
- Not a knowledge base per se — an agent memory infrastructure layer
- Adjacent to knowledge platforms: provides the memory primitives agents need
