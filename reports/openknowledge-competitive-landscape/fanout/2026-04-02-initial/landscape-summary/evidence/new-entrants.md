# New Entrants & Adjacent Tools Evidence

## Agent Memory Infrastructure (2025-2026 Launches)

### Zep / Graphiti
- **Website**: https://www.getzep.com/ — "Context Engineering & Agent Memory Platform"
- **arXiv Paper**: https://arxiv.org/abs/2501.13956 — "Zep: A Temporal Knowledge Graph Architecture for Agent Memory"
- **GitHub (Graphiti)**: https://github.com/getzep/graphiti — ~14,000 stars in 8 months, 35+ contributors, 25K weekly PyPI downloads
- **MCP Server**: https://www.getzep.com/product/knowledge-graph-mcp/ — Knowledge Graph MCP Server
- Key: Temporal knowledge graphs with validity windows per fact. 18.5% accuracy improvement, 90% latency reduction vs baselines.

### OpenViking (ByteDance / Volcengine)
- **GitHub**: https://github.com/volcengine/OpenViking — Open-sourced January 2026, 15,000+ stars, ~1,000 forks
- **Threads post**: https://www.threads.com/@sung.kim.mw/post/DU-CEXUldYK — "The Context Database for AI Agents"
- **Emelia Hub**: https://emelia.io/hub/openviking-context-database-ai-agents
- Key: File-system paradigm for agent context. 3-tier loading (L0/L1/L2). 95% cost reduction. 49% improvement on LoCoMo10. viking:// protocol.

### SuperLocalMemory
- **Website**: https://www.superlocalmemory.com/ — V3 with mathematical guarantees
- **GitHub**: https://github.com/qualixar/superlocalmemory — MIT license
- **arXiv**: Referenced in https://arxiv.org/html/2603.02240
- **Benchmark comparison**: https://dev.to/varun_pratapbhardwaj_b13/5-ai-agent-memory-systems-compared-mem0-zep-letta-supermemory-superlocalmemory-2026-benchmark-59p3
- Key: Local-first, zero-cloud. MCP + agent-native CLI. Fisher-Rao geometric retrieval. Mode A: 74.8% zero-cloud; Mode C: 87.7%. MIT license. EU AI Act compliant by design.

### Letta (formerly MemGPT)
- **Website**: https://www.letta.com/ — "The platform for building stateful agents"
- **GitHub**: https://github.com/letta-ai/letta
- **Blog**: https://www.letta.com/blog/memgpt-and-letta — MemGPT is now part of Letta
- Key: LLM-as-OS paradigm. Self-editing memory (core/archival/recall). Conversations API for shared memory. V1 architecture for GPT-5/Claude 4.5.

### Graphlit
- **Website**: https://www.graphlit.com/
- **GitHub**: https://github.com/graphlit
- **vs Mem0**: https://www.graphlit.com/vs/mem0
- **vs Letta**: https://www.graphlit.com/vs/letta
- Key: Cloud-native semantic memory. 30+ connectors. Multimodal (docs, audio, video, web). Automated entity extraction. Knowledge graphs across content types.
