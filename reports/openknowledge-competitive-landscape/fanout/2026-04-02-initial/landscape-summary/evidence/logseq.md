## Key Facts

- License: Fully free and open source (AGPL-3.0)
- Core: Block-based outliner with bidirectional links, graph view
- Features: Tasks, flashcards, whiteboards, Datalog queries, PDF annotation, Zotero integration
- Major 2025-2026 pivot: New SQLite-backed "DB version" replacing file-based system
- Original limitation: graphs >2,000 pages took 4-10 min to load; >300MB caused startup failures
- DB version adds Real Time Collaboration (RTC) — alpha as of early 2026
- New mobile app (iOS, Android coming)
- AI: Community plugins only (logseq-plugin-ai-assistant, AssistSeq). No official AI features.
- MCP server exists (community-built) for AI agent interaction with Logseq graphs
- Community forum thread asks about alignment with agentic AI trend — no official response
- Local Markdown or Org-mode files — no cloud, no vendor lock-in by default
